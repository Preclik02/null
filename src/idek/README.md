## IDEK

Idek is a custom lightweight text editor inspired by vim. Idek wsa built with main purpuse of being part of the ecosystem for NullOS (https://github.com/preclik02/NullOS). This project was originally made by preclik02 but was transfered under company Jaylubiny.

As you may know idek have been quite out-dated and the future developement of it was mainly on the official tool-set repository (null). With this commit the developement is officialy back in updated status. For the null it will need to install idek with cloning this repository so it will be fully wiped off there in the future.


## How to operate idek

When you first open idek you might don't know what to do and how to move and stuff like that, there is simple guide on how to do so. . ..

So after instalation you can open files by starting idek and putting in the filename, we are aware of a bug with inputing the filename that is invisible right now we are working on it.
After you open a file you might think you can start typing but that would be wrong. You have to put yourself into so called "insert" mode by pressing "a" on you keyboard, after you do so you can start typing.
If you are tired of pressing arrows on you keyboard to move in you code/file you can put yourself back into so called "normal" mode by pressing "esc" on you keyboard and move with "h, j, k, l" keys on your keyboard. When you are where you wanted to be you can again press "a" and start typing.
If you just opened your file and don't want to hold "j" to go down the file you can press "G" on your keyboard in normal mode to move the cursor to the bottom of your file the same goes from bottom to top with double pressing the "g" on your keyboard.
When you are done typing in your file you can press the ":" on your keyboard in normal mode to quit.
